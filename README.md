# Backup-Manager-Backend
[Nodemon](https://nodemon.io/) überwacht alle Änderungen im Quellcode und startet den Server automatisch neu; perfekt für die Entwicklung.  
Um den Server mit nodemon auszuführen:  
```bash
npm run dev
```
